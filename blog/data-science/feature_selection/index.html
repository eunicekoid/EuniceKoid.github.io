Feature Selection Methods

BLUF

The bottom line up front...

Feature selection methods use statistical techniques to select a subset of relevant features for machine learning from a larger set of features. The goal is to reduce dimensionality of the input data to enhance model performance, reduce overfitting, speed up training, and increase interpretability.

Filter-based Methods

About

Filter-based methods use statistical metrics or some criteria to assess the relevance of the features. Based on the metrics, the features are included or excluded. Various metrics that can be used depending on the context:

Continuous Variables:
▸ Correlation

Highly correlated features are removed.

Pearson Correlation Coefficient: \[ r = \frac{\sum{(X_i - \bar{X})(Y_i - \bar{Y})}}{\sqrt{\sum{(X_i - \bar{X})^2} \sum{(Y_i -\bar{Y})^2}}} \] where:
Xi and Yi are the individual data points for variables X and Y
X̅ and Y̅ are the means of variables X and Y, respectively
The numerator represents the covariance between X and Y
The denominator is the product of the standard deviations of X and Y


▸ Mutual Information

Select features with high mutual information with the target



Categorical Variables:
▸ Chi-Squared

Calculate the chi-square metric between the target and the numerical variable and only select the variable with the maximum chi-squared values. The intuition is that if a feature is independent to the target it is uninformative for classifying observations.
Chi-Squared Metric: \[ χ^2 = \sum \left( \frac{(O_i - E_i)^2}{E_i} \right) \] where:
Oi is the observed frequency for category i
Ei is the expected frequency for category i


▸ Mutual Information

Select features with high mutual information with the target


▸ Linear Discriminant Analysis (LDA)

LDA is more of a dimensionality reduction technique rather than pure feature selection. It finds linear combinations of features which are computed in such a way that the separation between the classes is maximized in the transformed feature space. In other words, it maximizes the ratio of between-class and within-class variance, ensuring that the distance between the means of the classes is maximized while the variance within each class is minimized.


▸ Analysis of Variance (ANOVA)

ANOVA determines if there is a significant difference between the means of two or more groups by examining the variance between groups to ascertain whether the means' differences are statistically significant. Select the features that have statistically significant differences in means, as this implies that the categories are distinct and impact the dependent variable. Assumes that the data is normally distributed and only works with one dependent variable and one categorical independent variable (could use two-way ANOVA for multiple independent variables).



Advantages

  • Faster and computationally more efficient than wrapper methods because you don't have to train additional models
  • Contributes to dimensionality reduction and potentially improved generalizability because of a simpler model
  • Reduced noise in the data
  • Easily scalable
  • Ranks the features by relevance, making it easier to select features and interpret their impact

Disadvantages

  • Does not address multicollinearity among features and assumes independence of features
    • Multicollinearity happens when two or more features are high correlated, which makes it tough for the model to delineate the individual contribution of each feature. Multicollinearity should be addressed prior to using feature-based methods.
    • Because filter-based methods assess features individually (univariate statistical tests), they do not consider the combined effect of highly correlated features. For example, some features may be important when combined with other features, but the univariate test does not capture this importance and discards the feature.
  • Does not consider the model performance, as features are selected before the model is trained. It does not adapt to the specifications of the machine learning algorithm.
  • Sensitive to scale of the features. Needs to be standardized, if necessary, beforehand.
  • Does not address imbalanced datasets.
