Principal Component Analysis (PCA)

BLUF (bottom line up front)

Principal Component Analysis is an unsupervised learning linear dimensionality reduction technique that finds the principal components in the data by linearly transforming the it into a new coordinate space. Principal components are linear combinations of the original features with the highest variance. Ultimately, this technique drops features with small variances and converts high-dimensional data into lower-dimensional representation while preserving as much variation from the original dataset as possible.

How It Works

  1. Calculate the average of two observations across all features
  2. Using the average values, calculate the center of the two observations
  3. Shift the dataset such that the center is at the origin (0,0)
  4. Fit a line through the points. The best fit line is one that maximizes the sum of squared distances from the points on the fitted line to the origin.
