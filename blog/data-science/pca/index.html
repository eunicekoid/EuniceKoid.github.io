Principal Component Analysis (PCA)

BLUF (bottom line up front)

Principal Component Analysis is an unsupervised learning linear dimensionality reduction technique that finds the principal components in the data by linearly transforming configurations of different features into a new coordinate space. Principal components are uncorrelated and represent the best linear combinations of the original features with the highest variance (information). Ultimately, this technique drops features with small variances and converts high-dimensional data into a lower-dimensional representation while preserving as much infornmation from the original dataset as possible.

How It Works

PCA uses Singular Value Decomposition (SVD) to find the principal components.

  1. The average measurement across all features is calculated and plotted along with the observations in the dataset.

  2. Using the average measurement, all of the points are shifted so that the center (average measurement) is aligned to the origin. All the other observations maintain the same relative distance from the center.

  3. The best-fit line that goes through the origin is determined by maximizing the sum of squared distances from projected points on the line to the origin. This is the principal component, the linear combination of the features.
    • Projected points on the fitted line are the points where the sum of squared distances between observations and the line are minimized.
    • The slope of the best-fit line (let's call it PC1) indicates the spread of the data. If the slope is greater than 1, that means the feature on the x-axis contributes more information to PC1 than the feature on the y-axis.

  4. Using the slope of PC1, the eigenvector and singular vector for PC1 can be determined.
    • The eigenvector is the unit vector that shows the direction of maximum variance.
    • The proportion of variance contributed by each feature to PC1 is indicated by the loading scores.
    • The average of the sum of squared distances of each projected point on PC1 to the origin is called the eigenvalue for PC1.
      • $$\Sigma \frac{distances^2}{n-1} = eigenvalue$$
    • The singular value of PC1 is the square root of the sum of squared distances of PC1.
      • $$\sqrt{\Sigma distances^2} = singular value$$
