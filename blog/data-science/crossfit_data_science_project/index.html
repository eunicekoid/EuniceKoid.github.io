CrossFit Data Science Project | Exploratory Data Analysis

Introduction

This article is part of a larger project to build a machine learning model that analyzes over 1 million observations of CrossFit athletes' benchmark workout data to predict ranking in the CrossFit Games.

Data was collected from the CrossFit Games website for the 2023 Games.

Athletes' profile data also has information on their affiliate gym, region, division (e.g., men, women), games level, qualifier (open, quarterfinals, semifinals, games), and rank.

Data Cleaning

The data was cleaned as follows:

  • Removed all athletes with no benchmark information
  • All weights were converted to pounds
  • Invalid and unreasonable values were removed (e.g., deadlifts of 1,000,000 lbs, 400m time of 20 seconds, negative values)
  • Time and weight values were standardized to a consistent format


Data Quality Issues

Imbalance
  • Division, Games_Level, and Qualifier are highliy imbalanced (52%, 80%, and 57%, respectively). This suggests that most competitors are in the Men's division, worldwide games level, and in the open qualifier. It makes sense that there would be more data worldwide and for the open, since that is the first level of the CrossFit games. Only those that qualify will advance to the higher levels.
Missing Values
  • Every benchmark workout has missing values. However, the following workouts have more than 50% missing data: Chad1000x, L1 Benchmark, Filthy 50, Grace, Helen, Run 5k, Sprint 400m, Fight Gone Bad, and Max Pull-ups. The lifts - Back Squat, Clean and Jerk, Deadlift, and Snatch - have the most complete data.

Correlations
  • The lifts are highly correlated with each other (back squats, deadlifts, clean and jerk, and deadlifts)
  • Fight Gone Bad is highly correlated with the clean and jerk, max pull-ups, L1 Benchmark, Filthy 50, Fran, Grace, and Helen. These workouts have common movements such as overhead presses, clean and jerks, deadlifts, pull-ups, burpees, and wall balls.
  • Max pull-ups is highly correlated with clean and jerk, Fight Gone Bad, snatch, and Fran. These workouts have common movements such as overhead presses and pull-ups.
  • Sprint 400m, Run 5k, and Helen are highly correlated. These workouts involve running.
  • Filthy 50, Fight Gone Bad, Fran, Grace, and Helen are highly correlated.

Dealing with Data Quality Issues

I keep only the Men's and Women's divisions, since the data is imbalanced with almost three-quarters of the observations falling in these two divisions. To predict rankings, I split the data by qualifier (open, quarterfinals, semifinals, and games).
