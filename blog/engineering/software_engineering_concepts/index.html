Describes the worst-case runtime scenario, the asymptotic upper bound of an algorithm.
 O(1)   Constant time. Example: retrieving a value from array with size N. 
 O(logN)   Log time (cutting N in half). Examples: binary search, heap pop, BST. 2x=n. x=log2n
 O(\(\sqrt{N}\))   Factors of N. 
 O(N)   Linear time. Examples: traversing through an array, heapify, sliding window, monotonic stack. 
 O(NlogN)   Log linear time. Examples: mergesort, heapsort. 
 O(N2)   Quadratic time. Examples: traversing an NxN grid, insertion sort, traversing through all pairs in a nested for-loop. 
 O(2N)   Exponential time. Examples: recursion, tree heigh of N with two branches (e.g., Fibonnaci). 
 O(N!)   Factorial time. Example: generating permutations of an array. 

Describes the best-case runtime scenario, the asymptotic lower bound of an algorithm

Describes both the upper and lower bound of a function, the average time complexity.
