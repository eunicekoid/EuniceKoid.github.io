Predicting CrossFit Games Ranking from Benchmark Workout Data with Random Forest and Neural Networks

Methods: random forest, principal component analysis, webscraping, neural networks

View code on Github


In this project, I scraped over 1 million records of CrossFit benchmark workout data to use as features to predict CrossFit Games ranking. I employ techniques and algorithms like principal component analysis, hyperparameter tuning, feature engineering, random forest, and neural networks.
